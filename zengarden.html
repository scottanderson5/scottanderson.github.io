CSS Zen Garden
Time Frame: 2.5 days
(See the project on GitHub or view it live)
Technology Used

RWD, "HTML", SCSS, Git.

Foundation

CSS Zen Garden is right of passage for many a web designer. The challenge: To create a beautiful and interesting web page without touching the HTML already provided. And the only language you are allowed to use is CSS/SCSS, a web designers best friend.

The exercise is a wonderful way to practice and perfect ones CSS skills. Using such items as "nth-child" and ":before" & ":after" becomes vital to completing the task.

Research

I began by looking at the HTML itself, thinking it best to explore what I had to work with before jumping into a plan. Indeed, the HTML was pretty sparce, and required some heavy thinking and skills to accomplish an appealing design.

I would need to think of a design as well. I perused some already completed zen gardens and decided that I needed to draw inspiration from a particular genre or artform, as some others had done.

Design

I decided to garner inspiration from Russian Constructivist artforms. There is an instant recognizability to the genre, and I knew that I could have fun putting something together.

I did some quick research and found common elements present so that I could think of potential colors, fonts, and images. I put some stile tiles together to help me explore these various possibilities. I wanted to get some ideas from elsewhere while keeping the design, ultimately, mine.
